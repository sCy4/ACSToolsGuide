GETTING STARTED

使用方式

01

啟動程式

左鍵雙擊桌面「哲盟枝椏」程式
(程式會安靜啟動，不會跳出任何視窗)。

02

選中目標欄位

選中哲盟系統中的目標欄位
(請見下方介紹)。

03

按 Win + Z

叫出功能選單，點選你要的功能。

壹

快速簽收

目標欄位：「簽收」視窗「運單號碼」欄位

0.5 秒快速簽收

  1. 複製單號
  2. 哲盟中開啟「簽收」視窗，點一下「運單號碼」輸入框。
  3. 按 Win + Z，在功能選單中「◇ 快速簽收」類型下點選你要的簽收類型。
影片 · 快速簽收操作示範
貳

快速問題

目標欄位：「問題件管理」視窗「運單號碼」欄位
影片 · 快速問題操作示範

1.8 秒快速錄入問題件

  1. 複製單號
  2. 哲盟中開啟「問題件管理」視窗，點一下「運單號碼」輸入框。
  3. 按 Win + Z，在功能選單中「◇ 快速問題」類型下點選你要的問題類型。
參

簽收監控人員捷徑

目標欄位：「簽收監控」視窗「派件員」欄位

0.6 秒指定簽收監控外務

  1. 開啟「簽收監控」視窗，點一下「派件員」輸入框。
  2. 按 Win + Z，在「◇ 簽收監控人員捷徑」類型下點選你要查看的人員。

貼上文本

可在任何欄位貼上文本
影片 · 貼上文本操作示範

在任何地方貼上你設定好的文字內容

  1. 滑鼠點一下想貼上文本的欄位（不限哲盟內）。
  2. 按 Win + Z，在「◇ 貼上文本」類型下選擇你要貼的文字。
CUSTOMIZE

編輯區：客製化自己的功能選單

四種功能類型的項目都是可以修改的。

進入編輯區

在螢幕右下角工作列中，左鍵點一下「哲盟枝椏」圖示，即可開啟編輯區。

開始編輯

點選編輯區最上方的分頁標籤可選擇功能類型；點選清單最下方的「＋ 點此新增」，於下方欄位輸入標題與內容即可新增項目；點選既有項目則可於下方欄位修改，按下「刪除此項」即可刪除項所選目；點擊 ▲ ▼ 按鈕調整項目順序。

確認才生效

所有變更要按「確認」後才會儲存並套用；按「本頁還原預設」可將此功能類型的項目回歸原廠設定。

你在編輯區設定的項目會儲存在你的電腦，與其他人互不影響。

CONFIG FILE

進階玩法：設定檔

I

什麼是設定檔

編輯區的項目會儲存在你電腦中的設定檔，設定檔與「哲盟枝椏」程式分開運作。可以如此理解：程式是個啟動器，用以啟動你的設定檔。

II

備份設定檔

備份設定檔可以讓你在不同電腦上使用同一組功能選單。你需要在檔案總管的路徑欄貼上這串路徑「%LOCALAPPDATA%/ACStools/哲盟枝椏.ini」再按 Enter 開啟你的本機設定檔，然後複製裡面文字，保存在你的記事本中。

III

使用別人的設定檔

你可以使用別人設定好的功能選單。只要在自己的本機設定檔中，把裡面文字覆蓋成別人設定檔中的文字即可。
